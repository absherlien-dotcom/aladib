# بوابة شبكة الأديب نت

واجهة عربية حديثة ومتجاوبة لتسجيل الدخول إلى شبكة **الأديب نت اللاسلكية**، مصممة لتعطي انطباعًا بالسرعة والأمان والاستقرار.

## المزايا

- إدخال رقم الكرت وتسجيل الدخول.
- رقم الكرت يُستخدم تلقائيًا كاسم مستخدم وكلمة مرور، بما يناسب أنظمة الكروت الشائعة.
- دعم متغيرات MikroTik Hotspot وCHAP.
- اختيار سرعة الإنترنت: اقتصادية، متوازنة، سريعة، فائقة.
- تبويب مستقل لتحديث السرعة، جاهز للربط مع API.
- مساحة بنرات إعلانية متحركة تلقائيًا.
- شريط إشعارات نصية متحرك.
- روابط الأسعار، نقاط البيع وخدمة العملاء.
- تصميم متجاوب للجوال والكمبيوتر.
- لا توجد مكتبات خارجية؛ الصفحة خفيفة وسريعة.

## الملفات

```text
index.html
assets/
  favicon.svg
  banner-fast.svg
  banner-safe.svg
  banner-offer.svg
```

## تركيبها على MikroTik

1. انسخ `index.html` ومجلد `assets` إلى مجلد `hotspot` في الراوتر.
2. تأكد من وجود ملف `md5.js` الافتراضي الخاص بـ MikroTik داخل مجلد `hotspot`.
3. افتح شبكة الواي فاي وجرب الدخول بواسطة رقم كرت فعّال.

> الصفحة تحتوي متغيرات MikroTik مثل `$(link-login-only)` و`$(error)`، ولذلك يكتمل عمل تسجيل الدخول الحقيقي عند وضعها داخل مجلد Hotspot في الراوتر.

## تعديل الإشعارات

في نهاية `index.html` ستجد:

```js
const CONFIG = {
  announcements: [
    'الإشعار الأول',
    'الإشعار الثاني',
    'الإشعار الثالث'
  ],
  speedUpdateEndpoint: ''
};
```

غيّر النصوص داخل `announcements` لتعديل الشريط المتحرك.

## استبدال البنرات

استبدل الصور الموجودة داخل مجلد `assets` مع المحافظة على الأسماء، أو عدّل المسارات داخل `index.html`:

- `assets/banner-fast.svg`
- `assets/banner-safe.svg`
- `assets/banner-offer.svg`

المقاس المقترح للبنر: **1200 × 525 بكسل** أو أي صورة بنسبة قريبة من `16:7`.

## ربط تحديث السرعة

ضع رابط خدمة تحديث السرعة في:

```js
speedUpdateEndpoint: 'https://example.com/api/speed'
```

وترسل الصفحة طلب `POST` بصيغة JSON:

```json
{
  "card": "رقم الكرت",
  "speed": "balanced"
}
```

قيم السرعة المتاحة:

- `economy`
- `balanced`
- `fast`
- `turbo`
